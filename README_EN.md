# DriftBottle

A global drift bottle social app that breaks language barriers through AI-powered real-time translation.

Every bottle you catch is a cross-cultural encounter.

## Core Features

- 🌍 **Cross-Cultural Exchange** — Connect anonymously with people around the world
- 🤖 **AI Real-Time Translation** — View both original and translated messages in chat bubbles
- 🏷️ **Topic Tags** — Meet like-minded people through shared interests
- 🎣 **Random Bottle Catching** — Experience the surprise of randomly discovering a message in a bottle
- 💬 **One-on-One Chat** — Reply to any bottle and start a real-time conversation
